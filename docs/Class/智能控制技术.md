# 智能控制技术

## 递阶控制

递阶控制系统由按照一定优先级和从属关系安排的子系统组成，这些子系统形成递阶结构，一般分为组织级、协调级和执行级。

- 执行级为直接控制级，其各个控制器分别控制被控对象的一部分；
- 协调级为监督控制级，其接受组织级的决策指令，协调和优化执行级各控制器的设定值，并向组织级传送执行结果信息；
- 组织级为决策控制级，其接受控制期望目标和协调级反馈信息，制定最优决策，对协调级下达决策指令。
